# SpaLush Green Color Scheme

## Primary Colors

### Main Gradient
- **Primary Green**: `#11998e` (Teal Green)
- **Secondary Green**: `#38ef7d` (Light Green)
- **Gradient**: `linear-gradient(135deg, #11998e 0%, #38ef7d 100%)`

### Hover State
- **Dark Teal**: `#0d7a6f`
- **Medium Green**: `#2dd46b`
- **Hover Gradient**: `linear-gradient(135deg, #0d7a6f 0%, #2dd46b 100%)`

## Text Colors

### Headings
- **Dark Green**: `#1a5f4f` (Primary headings)
- **Medium Gray**: `#2d3748` (Secondary text)

### Body Text
- **Gray**: `#4a5568` (Descriptions)
- **Light Gray**: `#9ca3af` (Placeholders)

## Background Colors

### Page Backgrounds
- **Gradient Background**: `linear-gradient(135deg, #11998e 0%, #38ef7d 100%)`
- **White**: `#ffffff` (Cards and containers)

### Form Elements
- **Light Green Background**: `#f0fdf4`
- **Border Color**: `#d4edda`

## Alert Colors

### Success
- **Background**: `#c6f6d5`
- **Text**: `#22543d`
- **Border**: `#9ae6b4`

### Error
- **Background**: `#fed7d7`
- **Text**: `#c53030`
- **Border**: `#fc8181`

## Shadow Colors

### Button Shadows
- **Default**: `rgba(17, 153, 142, 0.4)`
- **Hover**: `rgba(17, 153, 142, 0.6)`

### Card Shadows
- **Default**: `rgba(0, 0, 0, 0.15)`
- **Hover**: `rgba(17, 153, 142, 0.25)`

## Color Usage Guide

### Role Selection Page
- Background: Green gradient
- Cards: White with green accent bar
- Icons: Green gradient circles
- Buttons: Green gradient
- Checkmarks: Green gradient circles

### Signup/Login Pages
- Background: Green gradient
- Container: White
- Badge: Green gradient
- Labels: Dark green (#1a5f4f)
- Input borders: Light green (#d4edda)
- Input background: Very light green (#f0fdf4)
- Focus border: Teal green (#11998e)
- Submit button: Green gradient

### Spa Owner Dashboard
- Sidebar: Light green (#e8f0e8)
- Active menu: Green (#7fb77f)
- Stats cards: White
- Buttons: Green gradient
- Text: Dark green tones

## Accessibility

All color combinations meet WCAG 2.1 AA standards for:
- ✓ Text contrast ratios
- ✓ Interactive element visibility
- ✓ Focus indicators
- ✓ Error states

## Design Philosophy

The green color scheme represents:
- 🌿 **Nature & Wellness**: Spa and relaxation
- 💚 **Health & Vitality**: Fresh and energizing
- 🧘 **Balance & Harmony**: Calming and peaceful
- ✨ **Growth & Renewal**: Positive transformation

This color palette creates a cohesive, professional look that aligns perfectly with the spa and wellness industry while maintaining modern design standards.
